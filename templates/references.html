About

This data is currently in the Growth Charts API but will be moving to its own repository, which we envisage will become a repository for all growth references available in the public domain, internationally. We hope to cover other parameters such as blood pressure, and condition-specific grwoth charts such as Trisomy 21 (Down's Syndrome), Achondroplasia, and Turner's Syndrome.

Adding new content

To add a new Growth Reference or to let us know about missing types of growth references, or any other update we should include, please create an Issue in the International Growth References repository https://github.com/rcpch/growth-references/issues.

Pull requests are also welcome. At this stage, we are working hard to try to develop an internationally-usable serialisation format for all types of growth reference data, so please do **discuss with us** in advance how the data should be formatted before spending a lot of time on a PR!
